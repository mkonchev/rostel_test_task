# rostel_test_task
Тестовое задание ПАО Ростелеком КС

Парсер собирает данные о тарифах в частных и многоквартирных домах

## ИНСТРУКЦИЯ ПО ЗАПУСКУ

1) Клонировать репозиторий
```git clone https://github.com/mkonchev/rostel_test_task.git```

2) Если не установлен пакетный менеджер uv, то:
```pip install uv```

3) Если установлен(уже):
```uv sync``` Это должно установить зависимости

4) Создать .env файл в корне проекта рядом с .env.example(можно просто стереть .example)

5) Запустить
```uv run python main.py``` или ```python main.py```

6) ```.txt``` и ```.xlsx``` файлы появятся в директории output

7) Изучать тарифы кайфовать
